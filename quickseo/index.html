Quick SEO v3.0.0 is the all-in-one SEO plugin built for creators, developers, and store owners who want full control over their metadata—without limitations, paywalls, or fluff. This initial release includes meta box editing, schema management, canonical control, robots.txt editing, and broken link redirection—all natively inside WordPress.

Coming July 2025, version 3.1 will introduce full spreadsheet-style bulk editors for posts, pages, products, categories, and tags. Until then, Quick SEO is 100% free to download and use. In August 2025, the plugin will transition to a paid model—so grab it now and lock in early access while it lasts.

What You Get

Meta Box Integration

  • Custom Quick SEO meta box on all edit screens
  • Control SEO title, description, OG tags, Twitter tags, schema, and canonical URL
  • Injects meta into <head> and longtail block into page body (display:none)

Schema Editor

  • Edit WebPage, Article, Product, Offer, Person, and Service schema types
  • Save and manage JSON schema templates
  • Future updates will allow schema to be injected per post via spreadsheet view

Canonical / Redirects / Sitemaps / Robots

  • Set global canonical behavior or override per page
  • Scan for broken links and assign 301 redirects
  • Edit your robots.txt and regenerate your XML sitemap from inside WordPress

Pages / Posts / Products

  • Spreadsheet-style bulk editor coming soon in v3.1
  • Meta fields fully functional via the post editor meta box

Categories / Tags

  • Spreadsheet-style editor coming soon
  • Schema, OG, and longtail targeting support planned for v3.1+
