The Best Site Speed Tools for SEO (That Don’t Cost a Dime)

PageSpeed Insights (The Gold Standard)

Google’s own tool for measuring Core Web Vitals and page speed performance. Combines lab data (Lighthouse) and field data (Chrome UX Report). Great for benchmarking LCP, CLS, FID, and TBT. Use it to identify blocking resources, layout shifts, and slow load chains. Bookmark it. Test often. Fix what matters.

WebPageTest (Advanced Detail)

Free, open-source, and brutally detailed. Choose test locations, browsers, and connection speeds. See filmstrips, waterfall charts, and request-by-request breakdowns. Best for diagnosing edge cases or identifying exactly what’s causing a performance bottleneck. Use it alongside PSI for deep dives.

Bonus: run Lighthouse audits directly inside Chrome DevTools (Performance tab) for local development workflows.

Other Must-Haves in Your Stack

  • GTmetrix: Good visual overview. Mix of Lighthouse + proprietary metrics. Quick and useful.
  • Chrome DevTools: Built into your browser. Real-time audits, Core Web Vitals overlays, layout shift tracing.
  • SpeedVitals: Clean UI and actionable insights. Also helps benchmark mobile and FCP variations.
  • Network tab + Throttling: For developers—simulate 3G, see asset loads, spot JS/CSS bloat in real time.

Want to get serious? Self-host WebPageTest or roll your own testing script with Puppeteer + Lighthouse CLI.
